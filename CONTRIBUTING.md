## Getting Started

1. Fork the repository
2. Clone your fork
3. Open `OpenPhotoTool.xcodeproj` in Xcode
4. Create a branch for your feature/fix
5. Submit a pull request

## Development Setup

```bash
git clone https://github.com/openphototool/openphototool.git
cd openphototool
open OpenPhotoTool.xcodeproj
```

## Guidelines

- Write clear commit messages
- Follow Swift conventions and SwiftUI best practices
- Add tests for new features
- Keep PRs focused â€” one feature/fix per PR

## Reporting Issues

Use GitHub Issues. Include:
- macOS version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
